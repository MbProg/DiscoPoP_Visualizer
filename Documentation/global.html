JsonCuNode

A DiscoPoP Computational-Unit node in JSON format
Type:
  • Object
Properties:
Name Type Description
id nodeId The id of the node
type number The type of the Node (always 0)
start fileLine The first line of the CU's code-section
end fileLine The last line of the CU's code-section
childrenNodes Array.<Node> The nodes which are children of this CU (ids of the function-nodes being called)
parentNodes Array.<Node> The nodes of which this CU is a child (array with a single function or loop node id)
readDataSize number The amount of Bytes read in the CU
writeDataSize number The amount of Bytes written in the CU
successorCUs Array.<CuNode> The CUs succeeding this one
predecessorCUs Array.<CuNode> The CUs preceeding this one
basicBlockName string The name of the basic block of the CU
instructionsLineNumbers Array.<fileLine> The lines of instructions of the CU
readPhaseLineNumbers Array.<fileLine> The lines on which a read occurs within the CU
writePhaseLineNumbers Array.<fileLine> The lines on which a write occurs within the CU
localVariableNames Array.<Variable> The variables accessed locally by the CU
globalVariableNames Array.<Variable> The variables accessed globally by the CU
functionCall Array.<FunctionCall> The calls to functions from within the CU
RAWDepsOn Array.<Dependency> The Read-After-Write dependencies of the CU
WARDepsOn Array.<Dependency> The Write-After-Read dependencies of the CU
WAWDepsOn Array.<Dependency> The Write-After-Write dependencies of the CU
heatFactor number The node's heat in relation to all other nodes
